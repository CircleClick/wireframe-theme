
# HTML Template

### Setup

- Use `yarn install` from the command line to install dependencies.
- Build or watch the CSS/JS separately
    - `yarn js` or `yarn watch-js`
    - `yarn css` or `yarn watch-css`

- Run Compiler with Webpack --watch
